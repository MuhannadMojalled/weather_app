## Getting Started

In oreder to use the application, you must use your own api key.
In the file my_data.dart:

```
├───lib
│   ├───bloc
│   ├───data
│   │     └──my_data.dart
│   └───screens
```

Use your own api key:

```
String API_KEY = '<Api Key>';
```
